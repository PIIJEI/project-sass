Introduction
The purpose of this project

The main purpose of this project was to work on a practical self-initiated project based on a CSS pre-compressor called SASS (Syntactically Awesome Style Sheets). The main objective of the project was to create a functional 6 column layout grid system with additional features such icons, typography, navigation button styles and much more.

Currently the project is only a prototype and will be further developed including more features and responsive functions to work correctly on multiple devices.
