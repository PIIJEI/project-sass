Grid Layout

Would you like to know how to implement it in your site or why would you want to do that?
This is how to set it up.

First you have to include gridset.scss file in your sass components
Gridset.scss file has basic instructions of how to use it. You can build the grid by using classes with your own class names and implement it with the css cheat sheet that can assist you with the dimensions for each column.

You can also use an easier way to include the mixin that is prepared with the file archive. Those mixin include things like "gs-span", "gs-pad", "gs-float" and "gs-media". Those mixins will only work correctly if you include it with the source file gridset.scss.

How does the i.e gs-span work with the code?

Example:
HTML
<div class="className"> /* Content */ </div>
CSS
.className {
@include gs-span(lr, 1, 4)
}
